# UX 라이팅 가이드: 플러그링크의 말하기 원칙

플러그링크의 모든 텍스트에는 **명확함**, **따뜻함**, **간결함**이라는 세 가지 원칙이 적용됩니다.

## 왜 UX 라이팅 가이드가 필요한가

서비스 곳곳에서 사용자에게 말을 걸고 있습니다. 버튼 레이블, 에러 메시지, 안내 문구, 이메일까지. 이 모든 텍스트가 일관되지 않으면 사용자는 혼란을 느낍니다.

## 핵심 원칙

### 1. 명확하게

사용자가 한 번 읽고 이해할 수 있어야 합니다.

```
// Bad
처리가 완료되었습니다.

// Good
연결이 완료되었어요. 이제 서비스를 사용할 수 있어요.
```

### 2. 따뜻하게

기계가 아닌 사람이 말하는 것처럼 느껴져야 합니다.

```
// Bad
오류가 발생했습니다. 다시 시도해주십시오.

// Good
잠깐, 문제가 생겼어요. 다시 한번 시도해 주세요.
```

### 3. 간결하게

불필요한 단어는 과감히 제거합니다.

```
// Bad
현재 입력하신 비밀번호가 올바르지 않습니다. 다시 한번 확인 후 입력해 주시기 바랍니다.

// Good
비밀번호가 맞지 않아요. 다시 입력해 주세요.
```

## 가이드의 효과

UX 라이팅 가이드 도입 후, CS 문의 중 "사용법을 모르겠다"는 유형의 문의가 30% 감소했습니다. 좋은 텍스트는 좋은 디자인만큼 사용자 경험에 영향을 줍니다.
